# Module 3 — Programming with AI APIs

## Overview

This is the primary developer content. Module 3 covers the full implementation stack: SDK usage, prompt design patterns, function calling and tool use, and agentic loop patterns. All sections include working code examples in Java (Spring AI), Python (Anthropic SDK), and Node.js/TypeScript.

## Learning Objectives

By the end of this module, you will be able to:

- Make LLM API calls using Java Spring AI, Python Anthropic SDK, and Node.js @anthropic-ai/sdk
- Store prompt templates as versioned, testable code assets
- Apply five named prompt design patterns to real use cases
- Implement function calling with tool definitions, multi-turn tool loops, and failure handling
- Describe the ReAct pattern and explain when an agentic loop is and is not the right choice

## Prerequisites

- Module 1 (Foundations and Prompt Design)
- Programming experience in at least one of: Java, Python, or TypeScript/JavaScript
- Access to an Anthropic API key

## Primary Language

Java/Spring AI is the primary language for this module, matching the team's daily stack. Every section shows Java examples first. Python and Node.js equivalents follow where the pattern differs structurally — where the pattern is identical across languages, the prose says so and shows only the structural difference.
